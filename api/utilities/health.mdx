---
title: 'Health Check'
api: 'GET /healthz'
---

Check the health status of the Goanna API server.

## Response

<ResponseField name="status" type="string" required>
  Health status of the API (typically "ok")
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET http://localhost:8080/healthz
```
</RequestExample>

<ResponseExample>
```json 200 Response
{
  "status": "ok"
}
```
</ResponseExample>
