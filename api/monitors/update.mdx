---
title: 'Update Monitor'
api: 'PUT /v1/monitors/{monitorId}'
---

Update an existing monitor configuration.

## Path Parameters

<ParamField path="monitorId" type="integer" required>
  The unique identifier of the monitor to update
</ParamField>

## Request Body

<ParamField body="url" type="string" required>
  Target URL to monitor
</ParamField>

<ParamField body="cron" type="string" required>
  Cron expression for scheduling checks (e.g., "*/5 * * * *")
</ParamField>

<ParamField body="label" type="string">
  Optional label for the monitor
</ParamField>

<ParamField body="method" type="string" default="GET">
  HTTP method (e.g., GET, POST, PUT, DELETE)
</ParamField>

<ParamField body="iconUrl" type="string">
  Icon URL for the monitor
</ParamField>

<ParamField body="body" type="string">
  Request body content for POST/PUT requests
</ParamField>

<ParamField body="headers" type="object">
  Custom HTTP headers to include in requests (key-value pairs)
</ParamField>

<ParamField body="auth" type="object">
  Authentication configuration (key-value pairs)
</ParamField>

<ParamField body="notificationChannels" type="array">
  List of notification channels to use. Currently supports: `telegram`
</ParamField>

<ParamField body="selector" type="string">
  GJSON selector path for extracting values from JSON responses
</ParamField>

<ParamField body="expectedType" type="string" default="json">
  Expected response type: `json`, `html`, or `text`
</ParamField>

<ParamField body="expectedResponse" type="string">
  Expected response value to compare against the selector result
</ParamField>

<ParamField body="enabled" type="boolean" default="true">
  Whether the monitor should be enabled
</ParamField>

<ParamField body="triggerOnCreate" type="boolean" default="false">
  Whether to trigger a check immediately after update
</ParamField>

## Example Request

```bash
curl -X PUT http://localhost:8080/v1/monitors/1 \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://api.example.com/health",
    "cron": "*/10 * * * *",
    "label": "API Health Check (Updated)",
    "method": "GET",
    "expectedType": "json",
    "selector": "status",
    "expectedResponse": "healthy",
    "notificationChannels": ["telegram"],
    "enabled": true
  }'
```
