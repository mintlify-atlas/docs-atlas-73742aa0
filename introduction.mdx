# Welcome to Goanna

Goanna is a self-hosted web endpoint monitoring platform that helps you track the health and performance of your web services. Built with a modern stack, Goanna provides real-time monitoring capabilities with flexible scheduling and notification options.

## What is Goanna?

Goanna monitors your web endpoints by making HTTP requests on a scheduled basis and tracking their responses. Whether you're monitoring API health, checking specific JSON values, or tracking uptime, Goanna gives you the tools to stay informed about your services.

## Tech Stack

Goanna is built with:

- **Frontend**: TanStack Start + React + Tailwind + shadcn/ui
- **Backend**: Go API with Ent ORM
- **Database**: SQLite
- **Deployment**: Docker (single container for both services)

## Use Cases

- **API Health Monitoring**: Track the health of your REST APIs with scheduled checks
- **Uptime Monitoring**: Monitor website availability and response times
- **Data Change Detection**: Watch for changes in specific JSON values from your endpoints
- **Service Dependency Tracking**: Monitor third-party services your application depends on
- **Performance Monitoring**: Track response times and identify slow endpoints

## Architecture

Goanna uses a monorepo structure with:

- `apps/web` - Frontend application (TanStack Start + React)
- `apps/api` - Backend API (Go + Ent ORM + SQLite)
- `packages/api-client` - Generated TypeScript client from OpenAPI
- `openapi/openapi.yaml` - Shared API contract

The production deployment bundles both the web UI and API into a single Docker container for easy deployment and management.
